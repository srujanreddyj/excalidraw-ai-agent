# Excalidraw AI Agent

An AI-powered diagramming app that lets users generate, inspect, and modify Excalidraw diagrams through a chat-based agent.

## Stack

- React
- Excalidraw
- Cloudflare Workers
- AI SDK
- Upstash Vector
- Braintrust evals

## Setup

```bash
npm install
cp .dev.var
npm run dev
npm run build
npm run preview
npm run embed
npm run eval
```
